# OpenAI API Examples

Practical examples of using the OpenAI API for various tasks.

## Topics Covered

- Text completions
- Chat completions
- Content moderation
- Text categorization
- Speech to text (Whisper)

## Setup

1. Create a `.env` file with your API key:
```
OPENAI_API_KEY=your_key_here
```

2. Run the notebook cells

## Examples

- Company categorization
- Text classification
- Conversation handling
- Audio transcription

## Requirements

- OpenAI API key
- python-dotenv for environment variables
