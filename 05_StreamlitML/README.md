# Streamlit ML App

A web application for training and evaluating machine learning models interactively.

## Live Demo

[Open the app](https://portafolio-projects-guillermo-mares-galindo-cka9dg9ozfyxgsg3cs.streamlit.app/)

## Features

- Upload or use sample datasets
- Train Random Forest and SVR models
- View evaluation metrics
- Make predictions on new data

## Tech Stack

- Streamlit for the web interface
- scikit-learn for ML models
- pandas for data handling

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Files

- `app.py` - Main application code
- `housing.csv` - Sample dataset (California housing)
