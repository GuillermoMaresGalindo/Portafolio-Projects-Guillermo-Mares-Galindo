# Marketing Campaign Analysis

Analysis of a marketing campaign dataset to understand customer behavior and campaign effectiveness.

## Overview

This project explores marketing data to identify patterns in customer conversion, retention, and engagement across different channels.

## Key Topics

- Customer segmentation
- Conversion rate analysis
- A/B testing insights
- Channel performance comparison

## Data

The dataset includes customer demographics, campaign interactions, and conversion outcomes.

## Tools Used

- Python (pandas, numpy)
- Plotly for interactive visualizations
- Statistical analysis

## How to Run

1. Open `Notebooks/Marketing.ipynb` in Jupyter
2. Run cells sequentially
3. Data files are in the `Data/` folder

## Findings

Key insights from the analysis are documented within the notebook.
