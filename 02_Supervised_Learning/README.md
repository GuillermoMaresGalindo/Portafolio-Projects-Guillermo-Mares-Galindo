# Supervised Machine Learning with scikit-learn

A comprehensive introduction to supervised learning algorithms using scikit-learn.

## Overview

This notebook covers fundamental ML concepts and implementations, serving as a learning reference for classification and regression tasks.

## Topics Covered

- K-Nearest Neighbors (KNN)
- Linear Regression
- Train/Test Split
- Model evaluation metrics (accuracy, precision, recall, F1, RMSE)
- Overfitting and underfitting
- Cross-validation basics

## Datasets Used

- Telecom churn data
- Advertising budget and sales
- Diabetes dataset
- Spotify tracks

## How to Run

Open the notebook in Jupyter and run cells in order. All datasets are in the `data/` folder.

## Key Takeaways

- Understanding bias-variance tradeoff
- Importance of proper data splitting
- Metric selection based on problem type
